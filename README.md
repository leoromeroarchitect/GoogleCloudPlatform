# GoogleCloudPlatform — Learning Repository

Welcome to the GoogleCloudPlatform repository. This project aims to teach the tools and services of Google Cloud Platform (GCP) in a practical, organized way through guides, exercises, labs, and real projects.

## What is GCP?
GCP (Google Cloud Platform) is Google’s cloud services platform that provides infrastructure, managed platforms, and solutions for compute, storage, databases, networking, analytics, AI/ML, security, and more. This repository consolidates educational content to learn how to design, deploy, and operate applications and systems on GCP.

## Repository structure
- /docs — Theoretical guides and quick references  
- /labs — Step-by-step labs and automation scripts  
- /projects — Practical projects by level (with optional solutions)  
- /examples — Terraform, Kubernetes, IaC examples and deployments  
- /cert-prep — Resources and practice exams for GCP certifications

## Courses and descriptions
- Introductory Course (Level: Beginner)  
  - Content: cloud fundamentals, accounts, projects, IAM, billing, Cloud Console, and SDK.  
  - Goal: understand terminology and create your first GCP project and VM.

- Intermediate Course (Level: Intermediate)  
  - Content: Compute Engine, App Engine, Cloud Run, Cloud Functions, Cloud Storage, Cloud SQL, basic VPC.  
  - Goal: deploy web apps and serverless functions, manage storage, and simple networking.

- DevOps & Orchestration Course (Level: Intermediate–Advanced)  
  - Content: Google Kubernetes Engine (GKE), Terraform on GCP, CI/CD with Cloud Build and Artifact Registry, monitoring with Cloud Monitoring and Logging.  
  - Goal: automate deployments, manage clusters, and establish CI/CD pipelines.

- Data & Analytics Course (Level: Intermediate–Advanced)  
  - Content: BigQuery, Dataflow, Dataproc, Pub/Sub, Cloud Storage for data, ETL tools, and visualization.  
  - Goal: design data pipelines, run large-scale analytical queries, and architect data lakes/warehouses.

- AI/ML Course (Level: Intermediate–Advanced)  
  - Content: Vertex AI, AutoML, AI APIs (Vision, Speech, Natural Language), basic MLOps.  
  - Goal: train, deploy, and monitor ML models in production.

- Security & Architecture Course (Level: Advanced)  
  - Content: Identity & Access Management, VPC Service Controls, Cloud Armor, key and secret management, secure multi-zone architecture.  
  - Goal: design secure architectures and meet governance and compliance requirements.

- Hands-on Labs & Final Projects  
  - Content: guided exercises with step-by-step instructions, Terraform and Kubernetes templates, challenges to apply learned skills.  
  - Goal: consolidate abilities through real deliverables (APIs, microservices, data pipelines, scalable solutions).

- Certification Preparation  
  - Content: topic summaries by certification, practice questions, and mock exams (e.g., Associate Cloud Engineer, Professional Cloud Architect).  
  - Goal: prepare for official exams with a practical focus.

## Getting started
1. Clone this repository.  
2. Review /docs/intro.md to create your account, project, and set up the gcloud SDK.  
3. Follow the labs in /labs according to your level.  
4. Work on projects in /projects and use solutions as references.
